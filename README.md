## YouClone (Flask prototype)

Прототип видеохостинга в стиле YouTube с модульной архитектурой и заделом под развитие.

### Что уже есть

- регистрация/логин/логаут;
- загрузка видео с превью и базовой модерацией;
- домашняя лента, поиск и вкладка подписок;
- страница просмотра видео (просмотры, лайки/дизлайки);
- комментарии;
- профили авторов и подписки.

### Структура

- `app.py` — создание и конфигурация приложения;
- `auth.py` — маршруты авторизации;
- `routes.py` — web/API маршруты;
- `video_logic.py` — сервисный слой бизнес-логики (лента, реакции, комментарии);
- `models.py` — модели БД.

### Запуск

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
python app.py
```

Откройте: `http://127.0.0.1:5000/`.
