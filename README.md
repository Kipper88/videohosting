## Прототип видеохостинга (Flask)

Простой учебный прототип видеохостинга:
- загрузка видеофайлов;
- список загруженных видео;
- просмотр отдельного видео;
- лайки (счётчик на видео).

### Установка

1. Перейдите в папку проекта:

```bash
cd E:\projects_py\videohosting
```

2. (Опционально) создайте виртуальное окружение:

```bash
python -m venv venv
venv\Scripts\activate
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

### Запуск

1. Убедитесь, что вы в папке проекта:

```bash
cd E:\projects_py\videohosting
```

2. Запустите приложение:

```bash
python app.py
```

При первом запуске автоматически создастся база данных `app.db` и папка `uploads/` для хранения видео.

3. Откройте в браузере:

```text
http://127.0.0.1:5000/
```

### Как пользоваться

- Страница `/` — список загруженных видео.
- Страница `/upload` — форма загрузки:
  - введите название;
  - выберите видеофайл (mp4, webm, ogg, mov);
  - нажмите «Загрузить».
- Страница `/video/<id>` — просмотр конкретного видео и кнопка «Нравится», которая увеличивает счётчик лайков.
